# TradeIQ
TradeIQ is an AUREL-powered Apex/Tradovate trade analyzer that turns CSV exports into performance stats, strategy breakdowns, timing analysis, and actionable coaching.

Features
Upload Tradovate CSV exports
Review net P&L, win rate, profit factor, drawdown, and streaks
Tag trades by strategy
Compare strategy performance
Analyze timing by hour and day
Review entry and exit quality cards
Get an AUREL coaching summary based on imported trade data
Strategy Tags
TradeIQ supports these strategy tags:

ORB
VWAP
VWAP reclaim
Volume Profile
Orderflow
Trend continuation
Reversal
Breakout
Mean reversion
News play
Scalp
ICT/SMC
Other
Branding
Powered by AUREL, parent company of Market Maker Club.

How To Use
Open tradeiq.html in your browser.
Upload a Tradovate CSV file or load the demo dataset.
Review the dashboard tabs.
Tag trades in the All trades tab to unlock strategy analysis.
